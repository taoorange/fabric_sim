# Fabric Simulation 项目说明

## 1. 项目概述

这是一个基于 C++ 和 SFML 的实时布料模拟项目。程序通过质点与约束连接构建布料网格，结合 Verlet 积分和多次约束迭代，实现布料下垂、摆动、拖拽与撕裂等动态效果。项目现已扩展“小球发射”玩法：可蓄力发射弹丸，击中布料后打出洞口。

当前入口文件为 `main.cpp`，构建方式支持 CMake，且提供 `run.sh` 一键配置、编译和运行脚本。

> 说明：本项目在上游基础上的全部扩展功能（含小球相关功能）由 **Cursor 的 AI 功能** 参与开发实现。

---

## 2. 当前实现功能

### 2.1 布料网格初始化

- 使用二维网格生成布料点阵（`WIDTH x HEIGHT`）。
- 每个点保存 3D 坐标（`x, y, z`）。
- 顶行点固定（`locked = true`），形成“悬挂布料”效果。

### 2.2 物理更新（实时仿真）

- 采用 Verlet 方式更新粒子位置，不显式存储速度。
- 对每个点施加：
  - 重力（Y 方向下落）
  - 空气阻尼（减少能量振荡）
  - 简单风场（Z 方向正弦扰动）

### 2.3 约束求解（Link）

- 相邻点之间建立结构约束（横向、纵向）。
- 通过距离修正保持连接长度接近目标长度（`targetDist`）。
- 每帧进行多次迭代（当前为 8 次）提升布料“刚性”和稳定性。

### 2.4 撕裂与切割

- 约束拉伸超过阈值（`STRETCH_LIMIT`）时自动断裂。
- 右键拖动鼠标时，若鼠标轨迹与线段相交，则手动切断该约束。

### 2.5 交互操作

- 左键拖拽可抓取并移动布料点。
- 释放左键后恢复自由运动。
- 拖拽时会重置该点上一帧位置，避免“弹射”。

### 2.6 渲染显示

- 将 3D 点通过透视投影转换到 2D 屏幕。
- 使用 `sf::VertexArray` 绘制线框布料。
- 根据深度做颜色变化，并对抓取状态做高亮。

### 2.7 小球发射与打洞（新增）

- 支持“蓄力发射小球”交互（中键按住拉弓，松开发射）。
- 小球使用独立的 Verlet 运动更新，带重力与阻尼。
- 小球与布料 `Link` 发生碰撞时，触发命中。
- 命中后不只断开单条连接，而是按命中点半径批量断开周围连接，形成明显“洞口”。
- 小球支持发射瞄准线与弹丸可视化，便于观察力度和轨迹。

## 4. 核心算法说明

### 4.1 质点-约束模型（Mass-Spring / Stick Constraint）

- `Point` 代表质点
- `Link` 代表连接约束
- 布料由大量点和连接关系组成

### 4.2 Verlet 积分

核心思想：用“当前位置 - 上一帧位置”隐式表示速度，再推进下一帧位置。  
优点是实现简单、稳定性较好，适合此类实时布料模拟。

### 4.3 迭代约束求解

每帧多次遍历所有 `Link`，不断修正两端点距离误差，使布料结构更稳定、形变更自然。

### 4.4 小球碰撞与洞口生成

- 小球通过 `Ball` 结构管理，维护 `pos` 和 `prevPos`，采用 Verlet 方式推进。
- 碰撞检测使用“点到线段最短距离”判定（小球中心到布料连接线段的距离）。
- 当距离小于小球半径时，视为命中并记录命中点。
- 以命中点为圆心，在预设半径内批量断开布料连接，生成洞口。
- 命中后小球失活并回收，避免重复穿透同一区域。

在项目根目录执行：

```bash
./run.sh
```

脚本会自动执行：

1. CMake 配置（生成 `build`）
2. CMake 编译（生成 `build/fabric`）
3. 启动模拟窗口

## 6. 交互说明

- 左键拖拽：抓取布料
- 右键拖拽：切割布料连接
- 中键发射小球：按住蓄力，向后拉，松开发射
- 关闭窗口：结束程序

### 6.1 小球发射方法（详细）

1. 将鼠标移动到你希望的“发射起点”位置。  
2. 按住鼠标中键不放，进入蓄力状态（会看到瞄准线）。  
3. 保持中键按下，向“与目标相反的方向”拖动鼠标（即向后拉弓）。  
4. 拖动距离越长，蓄力越强，小球初速度越大（上限由 `BALL_MAX_PULL` 控制）。  
5. 松开中键后，小球会沿瞄准方向射出并飞向布料。  

### 6.2 发射方向与力度说明

- **方向**：由“锚点（按下中键的位置）到松开前鼠标位置”的反向决定。  
- **力度**：由后拉距离决定，距离越大，`x/y` 方向速度越大，同时 `z` 方向推进也更强。  
- **体感建议**：想打穿中心区域，可在布料中央前方按下，然后向屏幕下方或侧后方拉出一段再松开。

## 7. 关键参数（可调）

- 布料参数：`WIDTH`、`HEIGHT`、`DISTANCE`、`GRAVITY`、`AIR_FRICTION`、`STRETCH_LIMIT`
- 小球参数：`BALL_RADIUS`、`BALL_AIR_FRICTION`、`BALL_GRAVITY_SCALE`
- 发射参数：`BALL_MAX_PULL`、`BALL_FORCE_SCALE_XY`、`BALL_FORCE_SCALE_Z`
- 打洞参数：`HOLE_RADIUS`、`MAX_BALLS`
